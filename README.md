# War Card Game

A browser-based implementation of the classic **War** card game built with JavaScript and Vite.

## Overview

War is a simple card game where two players draw cards from the same deck. The player with the higher card wins the round and earns a point. The game continues until all cards have been drawn, and the player with the most points wins.

This project uses the Deck of Cards API to generate and draw cards dynamically.

## Features

* Shuffle a new deck at the start of each game
* Draw two cards per round
* Automatic score tracking
* Real-time round winner updates
* Final winner announcement
* Responsive card display

## Built With

* HTML5
* CSS3
* JavaScript (ES6+)
* Vite
* Deck of Cards API

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/war-card-game.git
```

Navigate to the project folder:

```bash
cd war-card-game
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
